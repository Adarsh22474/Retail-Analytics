# Retail Analytics Project

A concise, GitHub-friendly summary of the full Retail Analytics project.

## Overview

End-to-end retail analytics system using SQL + Power BI to uncover high-impact insights across sales, customers, products, stores, and retention. Includes complete data audit, modelling, dashboarding, and actionable business recommendations.

## What This Project Includes

* SQL Data Audit: Primary key checks, foreign key consistency, outlier detection, data cleaning.
* Data Modelling: Relationship building and transformation for analysis-ready tables.
* Power BI Dashboard: KPIs, sales trends, cohort analysis, store performance, customer satisfaction.
* Business Insights: Revenue drivers, retention issues, category performance, and operational improvements.

## Key Insights

* Clear upward sales trend with strong monthly and weekday patterns.
* Morning hours generate the highest sales volume.
* A small set of categories contribute a major share of revenue (Pareto behaviour).
* Strong performance differences across stores and states.
* Customer ratings highlight well-performing and weak regions.
* Retention drops significantly after the first month.

## Recommendations

* Improve 30‑day post‑purchase engagement to boost retention.
* Strengthen weak-performing stores with targeted measures.
* Prioritize inventory for top-performing categories.
* Offer weekday-specific promotions to lift low-traffic days.
